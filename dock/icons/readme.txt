How to extract an icon from an OSX app or folder

(For example from the folders in ../stacks)

https://www.idownloadblog.com/2014/07/22/howto-export-mac-icons-png-files/

1. Select the folder (or app)
2. Open the information window (CMD-i)
3. Select the icon in the left-top of the information window
4 .Copy (CMD-c)
5 .Open the preview app
6. Create a new document (CMD_n)
7. It should display the icon now
   If not, paste (CMD-v)
8. Export as an ICNS file (File > Export...)
